# Route map: Jarasandh killing

<hr/>

This route map is generated from the data returned by `/mb/v1/journeys/{name}` endpoint, with full details (`?expand=true`) where `{name}` is `Jarasandh killing`.

![Route map for Jarasandh killing](../images/map_jarasandh_killing.png)

The summary journey route is this:

```json
"journeyRoute": [
    "Indraprastha",
    "Kurujangal to Kalakuta",
    "Kalakuta to the east to Mithila",
    "Mithila to Magadh",
    "Girivraj",
    "Girivraj to Indraprastha"
  ]
```

The route details were taken by using the `?expand=true` query parameter with the journey name.

The following components were used to generate the map:

- Country boundary dataset from [Natural Earth](https://www.naturalearthdata.com/),  to determine the geographic extent of India and surrounding regions
- Geocoding by using [Nominatim](https://nominatim.org/)
- Terrain and physical basemap tiles from [Esri](https://www.esri.com/en-us/home), for the World Physical Map layer
